## Topics Covered
1. Course overview + the shell
2. Shell Tools and Scripting
3. Editors (Vim)
4. Data Wrangling
5. Command-line Environment
6. Version Control (Git)
7. Debugging and Profiling
8. Metaprogramming
9. Security and Cryptography
10. Potpourri
